# modexp
Copyright (c) 2020 Ronnie Song

This is a Rust based command-line calculator that computes *modexp()*.

<br>

*modexp()* is calculate the modular exponentiation:
```
modexp(x, y, m) = x ** y (mod m)
```

## Run

```
$ cargo run 2 20 17
   Compiling...
    Finished...
     Running...
The modexp of 2 ** 20 (mod 17) is 16
```
